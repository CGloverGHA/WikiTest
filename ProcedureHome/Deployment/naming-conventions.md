# Naming Conventions

The following types of applications should follow the following naming convention

| Application Type | Suffix      | Example                         |
| ---------------- | ----------- | ------------------------------- |
| API              | `.api`      | `license.api`                   |
| Service          | `.service`  | `apautomation.service`          |
| Frontend         | `.frontend` | `albertaquotingsystem.frontend` |
